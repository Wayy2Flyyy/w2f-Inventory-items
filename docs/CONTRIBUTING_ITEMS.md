# Contributing New Items

## Before you open a PR

- Follow naming standards in `docs/ITEM_NAMING.md`
- Include item definition fields used in this project
- Provide a matching image when applicable
- Run `python3 tools/validate_items.py`

## Submission checklist

- [ ] Item name is lowercase snake_case
- [ ] No duplicate keys in `ox_inventory/items.lua`
- [ ] `image` field matches actual image filename
- [ ] Descriptions are clear and professional
- [ ] README/docs updates included if needed
