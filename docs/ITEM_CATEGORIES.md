# Item Categories

This repository groups items into practical categories to help FiveM server owners build clear inventory systems.

## Hunting

Includes meat, pelts, trophy parts, and field tools.

Use this category for:
- Hunting jobs and progression
- Butcher/trader economy loops
- Wilderness gameplay systems

## Vehicle project

Includes shells, engine parts, body kits, and workshop components.

Use this category for:
- Project car progression
- Mechanic job crafting
- Performance upgrade systems

## Crafting materials

Includes scrap, wiring, plastics, and electronics.

Use this category for:
- Crafting benches
- Resource conversion loops
- Repair and production systems

## Food and drink

Includes basic consumables and quick-use snacks.

Use this category for:
- Hunger/thirst frameworks
- Shop inventory packs
- Roleplay restaurant or convenience stores

## Medical

Includes first-aid and trauma supplies.

Use this category for:
- EMS/medic gameplay
- Injury recovery systems
- Field treatment loops

## Economy and utility

Includes money-value items, access tools, and utility equipment.

Use this category for:
- Civilian progression
- Heist/police roleplay ecosystems
- General utility gameplay systems
