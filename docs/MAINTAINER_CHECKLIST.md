# Maintainer Merge Checklist

Use this checklist before merging pull requests:

- [ ] README still renders correctly
- [ ] Docs links work
- [ ] Lua syntax is valid
- [ ] JSON is valid (if changed)
- [ ] Item names use lowercase snake_case
- [ ] Image names use lowercase snake_case
- [ ] Image references use `.png`
- [ ] Manifest updated if needed
- [ ] Changelog updated if needed
- [ ] Validation script passes
- [ ] PR only changes relevant files
- [ ] No private data included
- [ ] No stolen or copyrighted assets included
